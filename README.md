# Front-End do sistema

Esse repositório serve como front-end do sistema de cadastro e georreferenciamento de telhados verdes na cidade do Recife, que está sendo desenvolvido como parte do plano de atividades de uma bolsa de pesquisa PIBIC do CNPq.
